# Changelog

All notable changes to Keyword Highlighter will be documented here.

## v1.1.0

- Added multilingual support: English, German, Spanish, French
- Removed color options from settings (not supported by PDF viewer)
- Cleaned up settings UI

## v1.0.0

- Initial release
- Highlight user-defined keywords in the Zotero PDF reader
- Ctrl+Shift+H shortcut
- Keyword categories with names
- Settings dialog via Tools menu
- Compatible with Zotero 7 and 8
